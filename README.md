# Highest GWA Finder

A Python program that reads a file containing 20 student names
and their General Weighted Average (GWA) separated by a comma,
then finds and displays the student with the highest GWA.

## What It Does
- Reads gwa.txt containing name and GWA pairs separated by a comma
- Compares all GWA values
- Prints the name and GWA of the top student

## Sample Input (gwa.txt)
Juan dela Cruz, 1.25
Maria Santos, 1.50
Jose Reyes, 1.75

## Sample Output
Highest GWA: Juan dela Cruz - 1.25

## Files
- gwa.txt — input file containing student names and GWA values

## Requirements
- Python 3.x
- A gwa.txt file in the same folder as the script
